Das Bundle beinhaltet zwei Funktionen:
1. Import der NPS/Kundenumfrage-Daten, um den Umfrage-Link an die aus Navision exportierten Kunden (CSV-Export) verschicken zu können. Die Funktion brauchen wir 2 mal um Jahr.
2. Import einer CSV-Datei (Export aus Navision), um den VB-Code anzupassen. So werden den Kunden also den E-Mail-Empfängern der korrekte Ansprechpartner im Vertrieb zugewiesen. Das brauchen wir nach entsprechenden Änderungen im Vertrieb (Neuausrichtung der Vertriebsgebiete durch Kündigungen, Neueinstellungen oder Neuorganisation). Nutzung bei Bedarf.
